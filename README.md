# React Redux with TypeScript 2

This project was bootstrapped with [Create React App](https://github.com/facebookincubator/create-react-app).

## Environment

* Node.js v8
* Yarn v1.5
* OS X 10.13.4


## Getting Started

```
git clone git@github.com:AtaruOhto/ns-redux-tutorial-2.git starter
cd starter
yarn 
yarn start
```
